Launch Your Environment in Minutes

Get your personal development cockpit up and running with three simple commands. All you need is a server with Docker.

  1. 01

    Clone the Repository

    Get the source code onto your server. This contains everything needed to build and run the application.

    git clone https://github.com/drmhse/ai-code-terminal.git
cd ai-code-terminal
  2. 02

    Configure Your Environment

    Create a `.env` file and add your GitHub OAuth credentials and the authorized username. This ensures only you can access the environment.

    cp env.example .env
nano .env # Edit with your favorite editor
  3. 03

    Launch with Docker

    Start the application using Docker Compose. It will build the container, set up the database, and launch the server, making it accessible on port 3014.

    docker-compose up -d
